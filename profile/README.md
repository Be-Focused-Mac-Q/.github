## 💡 Overview

<a href="#beFocused">Be Focused</a> implements the Pomodoro technique for Mac — a structured productivity method that alternates defined work intervals with scheduled breaks, maintaining concentration quality through regular rest rather than attempting open-ended uninterrupted focus that progressively degrades.

The <a href="#beFocused">Pomodoro session timer</a> runs configurable work intervals followed by corresponding short and long breaks — the classic model uses 25-minute work sessions with 5-minute short breaks and a longer 15-20 minute break after every four sessions, but Be Focused configures all durations independently to match individual concentration and recovery patterns. The structured work-rest alternation prevents the attention fatigue that extended uninterrupted work produces, maintaining higher effective concentration quality throughout longer working days than pure willpower-based focus attempts sustain. The <a href="#beFocused">task list with session assignment</a> ties each work session to a named task — tracking how many completed Pomodoros each task has accumulated, building an accurate record of where working time actually goes rather than where it was planned to go.

The <a href="#beFocused">daily goal setting</a> commits to a specific number of completed work sessions for the day — a concrete measurable productivity target with visual progress tracking throughout the day. The goal creates the kind of defined completion criterion that open-ended "work all day" intentions lack, and the visible progress toward the goal provides motivational feedback throughout the session. The <a href="#beFocused">session history and productivity statistics</a> record every completed work session with associated task, date, and duration — building a historical picture of productivity patterns over days, weeks, and months that reveals the time allocation reality versus the planned allocation.

<a href="#beFocused">Menu bar countdown display</a> shows remaining session time at a glance from any working context — the current timer visible without switching to Be Focused so the work session boundary stays visible while working in other applications. <a href="#beFocused">Break and session transition reminders</a> notify at the end of each work interval and break — the signal that structures the Pomodoro rhythm without requiring the user to monitor the clock.

<a href="#beFocused">iCloud sync across Apple devices</a> keeps tasks, settings, and session history consistent across Mac, iPhone, and iPad — the same task list and completed session record accessible on any device. <a href="#beFocused">Setapp availability</a> includes Be Focused in the Setapp subscription alongside other Mac productivity tools for subscribers who prefer consolidated billing.

## ❓ FAQ

<details>
<summary>What is the Pomodoro technique?</summary>

The Pomodoro technique alternates work intervals with scheduled breaks — typically 25-minute work sessions followed by 5-minute breaks, with a longer break after four sessions — maintaining concentration quality through regular rest.

</details>

<details>
<summary>Can I customize the timer interval lengths?</summary>

Yes. Work session duration, short break duration, long break duration, and sessions before a long break are all independently configurable.

</details>

<details>
<summary>Does Be Focused sync across Mac, iPhone, and iPad?</summary>

Yes. iCloud sync keeps tasks, settings, and session history consistent across Apple devices signed into the same Apple ID.

</details>

<details>
<summary>Is Be Focused available through Setapp?</summary>

Yes. Be Focused is available through the Setapp subscription service.

</details>

<details>
<summary>Does Be Focused track which tasks sessions are assigned to?</summary>

Yes. Each work session assigns to a named task and statistics show the session distribution across tasks.

</details>

<details>
<summary>Is there a free version of Be Focused?</summary>

Yes. A free version provides core timer functionality. The Pro version adds task tracking, detailed statistics, and additional features.

</details>

<details>
<summary>Does Be Focused show the timer in the Mac menu bar?</summary>

Yes. The current session countdown is visible in the menu bar from any Mac working context.

</details>

<details>
<summary>Can Be Focused set a daily session count goal?</summary>

Yes. Daily goal setting shows a configured target and tracks progress throughout the day.

</details>

<details>
<summary>Does Be Focused have an Apple Watch companion?</summary>

Yes. An Apple Watch companion shows current session status and recent session summary.

</details>
